**StatEngine Pro** is an enterprise-grade, full-stack analytical platform that bridges the gap between raw statistical calculation and formal academic publication. Designed specifically for researchers, statisticians, and data analysts, the application ingests raw data files, executes rigorous deterministic calculations, and generates mathematically validated, publication-ready **APA 7th Edition Results and Discussion sections** using advanced LLM reasoning.

---

## 🚀 Key Features

* **📂 Multi-Format Data Ingestion:** Drag-and-drop support for CSV, JSON, and native SPSS (`.sav`) data matrices. Includes interactive outlier detection using rolling $Z$-scores ($Z > \pm3$).
* **📐 Deterministic Execution Engine:** Offloads all statistical calculations to robust Python libraries (`scipy`, `statsmodels`, `pingouin`). **Zero mathematical hallucination.**
* **🧠 Deep Reasoning Synthesis:** Leverages Gemini (with high-thinking reasoning models enabled) inside Google AI Studio to interpret statistics and write expert methodological analysis.
* **📊 Vector-Grade Visuals:** Native interactive charting using `recharts`—generate publication-quality scatter plots, regression lines, and bar charts complete with Standard Error (SE) bars and significance brackets.
* **💾 Unified Export Center:** One-click downloads for high-fidelity PDFs, academic Markdown files, and clean diagnostic Excel files.
